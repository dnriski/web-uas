Menyisipkan Audio dan Video di Html

Oleh Jane Doe
Sebelum HTML5 diperkenalkan, elemen audio di web biasanya ditambahkan menggunakan aplikasi eksternal seperti Flash Player. Namun, dengan HTML5, kita kini memiliki tag bawaan yaitu <audio>. Tag <audio> digunakan untuk menyisipkan pemutar audio langsung di halaman web. Untuk menentukan file audio yang ingin diputar, kita dapat menggunakan tag <source/> di dalam tag <audio>.

Berikut ini adalah penerapan kode nya :

 
        <audio controls> 
            <source src="audio.mp3" type="audio/mpeg">
            Browsermu tidak mendukung tag audio
        </audio>

Untuk menyisipkan file video, Anda dapat menggunakan tag <video> dalam HTML5. Di dalam tag <video>, Anda bisa menambahkan tag <source>untuk menentukan lokasi file video dan formatnya dengan atribut src dan type. Tag <video>

Berikut ini adalah penerapan kode nya :

 
        
        <video controls>
        <source src="video.mp4" type="video/mp4" />
        Browsermu tidak mendukung tag ini
        </video>
